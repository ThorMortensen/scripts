# Thor's Quality Of Life scripts

#### To install

Make sure you have Git and Ruby installed.
```
sudo apt update && sudo apt install ruby && sudo apt install git
```


Add path in `.bashrc`.
```
PATH=$PATH:~/scripts/
```
Run chmod-script to make scripts executable.
```
__chmodScripts
```
